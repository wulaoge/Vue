# Vue
Vue环境搭建
1.安装Node.Js(`http://nodejs.cn/download/`)
  --检查安装：`cmd-->node -v  -->npm -v`
2.基于淘宝npm镜像安装相关依赖 `npm install -g cnpm –registry=https://registry.npm.taobao.org`，回车等待
3.安装vue-cli脚手架 `npm install -g vue-cli`，回车等待
4.用vue-cli创建项目
*cd desktop 进入桌面
*vue init webpack firstVue 创建项目
$cd desktop/firstVue
$cnpm install
$cnpm run dev
